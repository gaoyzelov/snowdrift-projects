· Capabilities ·

核心能力

精心设计的企业级功能，每个模块开箱即用。

📦

统一响应

Result<T> 统一 API 返回格式，ResultCode 结合 i18n 实现国际化状态码，BizException 自动解析为多语言错误消息。

🌐

模块化 i18n

每个模块独立维护国际化资源，请求级语言切换，响应消息自动解析。支持中英文，轻松扩展更多语言。

☁️

多端 OSS

策略模式驱动，一套 API 适配 5 种对象存储。批量删除、签名 URL、分片上传。配置即切换，支持运行时热更新。

🔒

分布式锁 & 防重

@DistributedLock + @RepeatSubmit 声明式注解，SpEL 动态 key，看门狗自动续期，防重复提交支持自定义时间窗口。

⏱️

动态调度

统一 IScheduleService 泛型接口，Quartz 和 XXL-JOB 双实现。运行时动态任务 CRUD，无需重启应用。

🛡️

安全双模

Sa-Token 和 Spring Security 双实现共用 ISecurityService 抽象，@AnonymousAccess 标记公开接口，TokenStore 支持 Redis/内存。

📝

ORM 增强

MyBatis-Plus 插件体系：多租户自动隔离、@DataScope 数据权限（5 种范围）、AES-GCM 字段透明加解密、自动填充审计字段。

🛡️

XSS 防护

可插拔 XssCleaner 接口，默认 HTML 实体转义零依赖。支持注入 Jsoup 等实现，覆盖参数、请求头、QueryString 全入口。

📨

消息队列

Spring Cloud Stream 封装，Kafka / RocketMQ / RabbitMQ 三 binder。@MqListener 自动注册 Consumer、拦截器链、HMAC 签名。

🔗

RPC 上下文传播

Dubbo 过滤器链自动传播 traceId + SecurityContext，Provider 端 finally 清理。异常智能序列化兼容，调用日志记录。

快速开始

克隆项目

从 GitHub 获取最新代码

Shell

02

编译构建

全量编译（跳过测试）

Shell
mvn clean install -DskipTests
03

引入模块

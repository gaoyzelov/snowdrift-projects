Snowdrift

Spring Cloud 微服务脚手架

基于 Spring Boot 3.5 + Spring Cloud 2025 的现代化企业级脚手架
提供插件化对象存储、完整国际化支持、统一异常处理等核心功能

🚀 GitHub 仓库 📖 快速开始

核心特色

精心设计的企业级功能，助力快速开发

🔌

插件化 OSS 存储

基于策略模式和工厂模式设计，支持本地存储、MinIO、阿里云 OSS 等多种存储方式，可动态切换和热更新配置

🌍

完整国际化支持

内置 I18n 国际化方案，所有异常消息和系统提示支持多语言，自动解析浏览器语言偏好，支持中英文切换

⚠️

统一异常处理

全局异常拦截器覆盖 15+ 种异常类型，统一返回格式，所有异常消息支持国际化，提供友好的错误提示

🔐

上下文管理

提供 HTTP 上下文和安全上下文管理，自动捕获请求信息（IP、User-Agent 等），支持用户信息线程隔离存储

⭐

开源共建

项目完全开源，采用 Apache 2.0 协议，欢迎提交 Issue 和 PR，一起参与建设，让脚手架更加完善

📚

代码规范

严格遵循命名规范和设计模式，统一的代码风格，完善的注释文档，便于团队协作和后期维护

模块架构

清晰的模块划分，便于理解和扩展

📦
snowdrift-common
已完成

通用基础模块，提供项目所需的核心工具类和基础设施

  • 统一返回结果封装（Result）
  • 业务异常基类（BizException）
  • 11 个常用工具类
  • 通用枚举和常量定义
🌐
snowdrift-web
已完成

Web 配置模块，提供完整的 Web 层基础设施

  • 全局异常处理器
  • 国际化支持（I18n）
  • HTTP 上下文过滤器
  • JSON 序列化配置
  • 静态资源映射
☁️
snowdrift-oss
已完成

对象存储模块，插件化设计支持多种存储方式

  • 统一操作接口（IOssService）
  • 策略工厂管理（OssStrategyFactory）
  • 本地存储实现（Local）
  • 支持分片上传
  • 支持热更新配置
📋
snowdrift-log
已完成

日志记录模块，基于 AOP 实现灵活的日志功能

  • API 日志注解（@ApiLog）
  • 登录日志注解（@LoginLog）
  • 日志服务接口
  • 异步日志处理
🔒
snowdrift-security
开发中

安全模块，集成认证和授权功能（计划集成 Sa-Token）

  • 用户认证
  • 权限校验
  • 安全上下文管理
⚡
snowdrift-cache
开发中

缓存模块，提供分布式缓存解决方案（计划集成 Redisson + JetCache）

  • 分布式缓存
  • 缓存注解支持
  • 分布式锁

快速开始

三步即可启动项目

01

克隆项目

从 GitHub 克隆项目到本地

git clone https://github.com/gaoyzelov/snowdrift-projects.git
02

编译构建

使用 Maven 编译项目

mvn clean install
03

运行测试

启动 sample 模块进行测试

cd snowdrift-sample
mvn spring-boot:run
